Create the config table

The monitoring config table was not found. One-time setup:

  1. In make.powerapps.com → Tables, create table Monitored Agent with schema name pda_monitoredagent.
  2. Add Choice column pda_agenttype: Copilot Studio Agent = 100000000, Cloud Flow = 100000001.
  3. Add Text column pda_targetid — paste the botid or workflowid GUID.
  4. Add Text column pda_embedurl — paste the Copilot Studio canvas URL to enable the embedded chat sidebar (optional).
  5. Create table Agent Run (pda_agentrun) with pda_agentid (Text), pda_starttime (DateTime) and pda_status (Choice: Running = 100000000, Completed = 100000001) — chat sessions are tracked here.
  6. Add one row per agent. Deactivate a row to stop monitoring it.

Different column names? Edit the CONFIG block in this file's script.
